HTML

What is the difference between your <head> and <body> tags?

This is an answer The head tag holds the meta data like link to css and title -things that you can't see. The body tag holds things that you can see- page content.

What are the main tags that go in the <head>?

  • Meta:things that sites read to identify what the purpose of the site is. Makes site responsive- viewport
  • Title: on the tab of the browser
  • Link: to link css to the page

What is the <meta> viewport tag for? Why is it important?

meta name="viewport" content="width=device-width, initial-scale=1" helps with screen resizing for mobile screens- don't need to know the exact properties on this tag, do need to know what it's used for.

How do you link a CSS file in your HTML?

< link rel="stylesheet" href="style.css" type= "text/css" >

How do you add an image to your HTML?

What are the three different <a> types in HTML?

  • internal link to another page in same site
  • external link to a different site
  • anchor link (#description)

What property is used on the <a> tag to open a link in a new tab? Write out a link to Google using the property to open in a new tab.

What are <div>and <span> tags? How are they used?

<div> tags are used as containers/wrappers -- is a block element. Used to group and identify classes in css. <span> tags are used for inline containers

How do you add a border to an image? What about margins?

                img {
                border: 1px solid black;
                border: [width][style][color]
                }

What are the three ways you can format padding (and margin) values when defining different sides of an element?

                img {
                padding: 10px; // applies 10px padding to all sides //
                padding: 5px 10px; // top/bottom then left/right //
                padding: top right bottom left; // defines individual sides//
                }

What does `margin: 0 auto;` mean with reference to the question above?

No margin on the top and bottom. Right and left are auto. Used to center content on a page. (also needs width and display- block)

What is a hexadecimal code? How is it used?

Used for colors (#000000) in css

What is a media query? Why is it used? Write a media query that changes the background color of your html page on small screen sizes.

makes page responsive. Changes the style of a page element/class/id at a defined page size. Could be min-width or max-width (could be both). 

                @media screen and (max-width: 500px) {
                  body{
                    background-color: green;
                  }
                }

What is flexbox? Why is it used?

Flexbox is a set of css properties and values that are used to move elements within a container. Useful for layout of a page.

cascading stylesheet. Styles written at the top can be over written by styles at bottom. specificity helps with this.
